# 📚 Multi-Subject Quiz Application

Multi-Subject Quiz Application is a console-based program built in C++ to test users’ knowledge in C++, Java, and HTML. It allows users to take randomized quizzes in different subjects, tracks scores, calculates percentages, and provides pass/fail results.

This project is ideal for beginners looking to improve programming skills while creating an interactive learning tool.

# 🔹 Features

Multiple-choice questions across three subjects:

C++

Java

HTML

Randomized questions for each quiz attempt

Immediate feedback for correct/incorrect answers

Marks calculation and percentage display

Pass/Fail status based on score

User input for name and roll number

Option to retake quiz

# 🔹 How It Works

User enters their name and roll number.

User selects a subject to take the quiz in.

Six randomized questions are displayed one by one.

User inputs their answer for each question.

At the end, the program displays:

Name and roll number

Total marks

Percentage

Pass/Fail status


# 🔹 How to Use

Clone the repository:

git clone <your-repo-link>

Open the project in a C++ IDE (like Code::Blocks, Dev C++, or Visual Studio).

Compile and run main.cpp.

Follow on-screen instructions to take the quiz.
